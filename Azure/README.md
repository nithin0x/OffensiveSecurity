# Azure Offensive Security Scripts

This directory contains offensive security scripts targeting Microsoft Azure environments.

## Overview

Scripts in this folder are designed for security assessments, penetration testing, and red team operations against Azure infrastructure.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [PowerShell/](PowerShell/) | PowerShell scripts for Azure security assessments |
| [Bash/](Bash/) | Bash scripts for Azure security assessments |
| [Python/](Python/) | Python scripts for Azure security assessments |

## Common Use Cases

- Azure AD enumeration and exploitation
- Managed Identity abuse
- Storage account misconfiguration detection
- Azure Function analysis
- Key Vault secrets extraction
- Virtual Machine reconnaissance
- Resource group and subscription enumeration
- Azure DevOps pipeline exploitation

## Prerequisites

- Azure CLI configured with appropriate credentials
- Az PowerShell module (for PowerShell scripts)
- Azure SDK for Python (for Python scripts)

## Disclaimer

These scripts are intended for authorized security testing only. Always obtain proper authorization before performing any security assessments.

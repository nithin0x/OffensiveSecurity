# Active Directory Bash Scripts

This directory contains Bash scripts for Active Directory security assessments from Linux systems.

## Scripts

*Scripts will be added here as they are developed.*

## Usage

```bash
# Example usage
chmod +x script-name.sh
./script-name.sh [options]
```

## Requirements

- Bash 4.0+
- LDAP utilities (ldapsearch, etc.)
- Kerberos utilities (kinit, klist, etc.)
- Impacket tools (optional)

# Web Application Bash Scripts

This directory contains Bash scripts for web application security assessments.

## Scripts

*Scripts will be added here as they are developed.*

## Usage

```bash
# Example usage
chmod +x script-name.sh
./script-name.sh [options]
```

## Requirements

- Bash 4.0+
- curl or wget
- jq (for JSON parsing)
- Common security tools (nmap, nikto, etc.)

# Web

This folder contains web application enumeration and exploitation helpers. Content typically includes automation scripts for discovery, parameter fuzzing, authentication bypass patterns, and data exfiltration proofs-of-concept.

## Safety & Legal
Only test against targets you own or have explicit permission to test. Many web scans are intrusive and may be blocked or interpreted as attacks.

## Contributing
PRs should include a description of the testing environment and any resources required to validate the script.

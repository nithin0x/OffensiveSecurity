# Web Application Offensive Security Scripts

This directory contains offensive security scripts targeting web applications.

## Overview

Scripts in this folder are designed for security assessments, penetration testing, and bug bounty hunting against web applications.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [PowerShell/](PowerShell/) | PowerShell scripts for web security assessments |
| [Bash/](Bash/) | Bash scripts for web security assessments |
| [Python/](Python/) | Python scripts for web security assessments |

## Common Use Cases

- Subdomain enumeration and reconnaissance
- Directory and file discovery
- SQL injection detection and exploitation
- Cross-Site Scripting (XSS) testing
- Server-Side Request Forgery (SSRF) detection
- Authentication bypass techniques
- API security testing
- JWT token analysis
- Web application fingerprinting
- Vulnerability scanning automation

## Prerequisites

- HTTP clients (curl, wget, requests)
- Web security tools (Burp Suite, OWASP ZAP)
- Python libraries (requests, beautifulsoup4, selenium)

## Disclaimer

These scripts are intended for authorized security testing only. Always obtain proper authorization before performing any security assessments.

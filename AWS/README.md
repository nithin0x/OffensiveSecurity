# AWS Offensive Security Scripts

This directory contains offensive security scripts targeting Amazon Web Services (AWS) environments.

## Overview

Scripts in this folder are designed for security assessments, penetration testing, and red team operations against AWS infrastructure.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [PowerShell/](PowerShell/) | PowerShell scripts for AWS security assessments |
| [Bash/](Bash/) | Bash scripts for AWS security assessments |
| [Python/](Python/) | Python scripts for AWS security assessments |

## Common Use Cases

- IAM enumeration and privilege escalation
- S3 bucket misconfiguration detection
- Lambda function analysis
- EC2 instance reconnaissance
- Secrets Manager and Parameter Store extraction
- CloudTrail log analysis
- VPC and security group enumeration

## Prerequisites

- AWS CLI configured with appropriate credentials
- Boto3 (for Python scripts)
- AWS Tools for PowerShell (for PowerShell scripts)

## Disclaimer

These scripts are intended for authorized security testing only. Always obtain proper authorization before performing any security assessments.

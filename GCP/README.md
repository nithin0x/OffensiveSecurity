# GCP Offensive Security Scripts

This directory contains offensive security scripts targeting Google Cloud Platform (GCP) environments.

## Overview

Scripts in this folder are designed for security assessments, penetration testing, and red team operations against GCP infrastructure.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [PowerShell/](PowerShell/) | PowerShell scripts for GCP security assessments |
| [Bash/](Bash/) | Bash scripts for GCP security assessments |
| [Python/](Python/) | Python scripts for GCP security assessments |

## Common Use Cases

- IAM enumeration and privilege escalation
- Cloud Storage bucket misconfiguration detection
- Cloud Functions analysis
- Compute Engine reconnaissance
- Secret Manager extraction
- Service account key exploitation
- VPC and firewall rule enumeration
- GKE cluster security assessment

## Prerequisites

- Google Cloud SDK (gcloud CLI) configured with appropriate credentials
- Google Cloud Client Libraries for Python (for Python scripts)

## Disclaimer

These scripts are intended for authorized security testing only. Always obtain proper authorization before performing any security assessments.

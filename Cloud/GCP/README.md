# GCP

This folder contains Google Cloud Platform (GCP) specific scripts, guides, and examples for enumerating and attacking misconfigurations in GCP projects.

Quick start
- Use `gcloud auth login` and set the project: `gcloud config set project <PROJECT_ID>`.

Common techniques
- Service account enumeration and key misuse
- IAM role misconfigurations and identity chaining
- Cloud Storage (GCS) enumeration and exfiltration

Safety & Legal
Only use these techniques in authorized test projects or labs.

# Azure

This folder contains Azure-specific scripts, guidance, and examples for offensive operations focused on Azure resources, identities, and services.

Quick start
- Use the `az` CLI with a test subscription: `az login` and `az account set --subscription <id>`.

Common techniques
- Azure AD enumeration (users, groups, roles, applications)
- Service Principal and Managed Identity misuse
- Role assignments and RBAC review for privilege escalation
- Storage and blob exfiltration patterns

Safety & Legal
